
### Creating an array [[Introduction]]
To create an array we need to use **np.arrray([ ])** in numpy  

```python
arr = np.array([10,20,30,40,50])
print(arr)

#OUTPUT
[10 20 30 40 50]
```


**To find out the length , dimension and shape of the array** 

```python
	  arr.ndim ------> # 1
	  
	  len(arr) -------> # 5
	  
	  arr.shape -------> # (5,)
```


### Creating a 2D array 
```python
arr_2d = np.array([
```
